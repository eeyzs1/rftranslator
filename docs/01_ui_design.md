# UI 设计方案 — rftranslator

> **注意**：本文档描述当前已实现的 UI 设计。以源码为准。

## 1. 设计原则

| 原则 | 说明 |
|------|------|
| Real Free | 无广告、无订阅、无追踪，完全离线 |
| 双端一致 | 核心视觉语言统一，交互细节按平台习惯适配 |
| 轻量本地 | 无网络依赖，UI 反馈即时 |
| 可读性优先 | 释义、例句字号层次清晰，适合长时阅读 |

## 2. 色彩系统

```
主色 (Primary)       #E8002D   默认红色（可从 12 种预设色中选择）
主色深 (Dark)        #B8001F
主色浅 (Light)       #FF4D6A
背景 (Background)    #F5F5F5   浅灰背景
卡片背景             #FFFFFF
分割线               #E0E0E0
文字-主              #1A1A1A
文字-次              #666666
文字-辅              #999999
音标                 #E8002D
词性标签背景         #FFF0F0
词性标签文字         #E8002D
例句高亮             #E8002D
收藏激活             #FFB800   金黄
```

### 主题色预设

12 种预设主题色，默认为 `#E8002D`（红色），用户可在设置中切换。

### Dark Mode

Android 和 Windows 均支持浅色/深色/跟随系统三种模式。

```
背景                 #121212
卡片背景             #1E1E1E
文字-主              #EFEFEF
文字-次              #AAAAAA
分割线               #2C2C2C
主色（暗色下）       #FF4D6A   略亮，保证对比度
```

## 3. UI 风格

应用支持三种 UI 风格，在设置中切换：

| 风格 | 导航方式 | 适合平台 |
|------|---------|---------|
| **Material 3** | 底部 NavigationBar | 移动端 |
| **Fluent Design** | 侧边 NavigationRail | 桌面端 |
| **自适应** | 根据平台自动选择 | 全平台 |

自适应模式下：桌面端自动使用 Fluent，移动端使用 Material 3。

## 6. 组件规范

### 搜索框

```dart
// 高度: 48dp, 圆角: 24dp
// 背景: #FFFFFF, 阴影: elevation 2
// 内边距: 水平16dp
// 光标颜色: 跟随主题色
```

### 词性标签 (PartOfSpeechChip)

```
背景: #FFF0F0, 圆角: 4dp
内边距: 水平8dp 垂直2dp
字号: 12sp, 颜色: 跟随主题色, 字重: Medium
```

### 收藏按钮

```
图标: star_border (未收藏, #999999)
      star (已收藏, #FFB800)
```

## 7. 空状态设计

| 场景 | 文案 |
|------|------|
| 无搜索结果 | "未找到结果" |
| 收藏为空 | "还没有收藏" |
| 历史为空 | "暂无翻译记录" |
| 词典未安装 | 引导下载词典 |
| 模型未安装 | 引导下载翻译模型 |
