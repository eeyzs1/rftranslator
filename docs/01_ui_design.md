# UI 设计方案 — 有道词典 Flutter 复刻版

## 1. 设计原则

| 原则 | 说明 |
|------|------|
| 忠实原版 | 色彩、排版、图标风格贴近有道词典 |
| 双端一致 | 核心视觉语言统一，交互细节按平台习惯适配 |
| 轻量本地 | 无网络依赖，UI 反馈即时 |
| 可读性优先 | 释义、例句字号层次清晰，适合长时阅读 |

词性标签文字         #E8002D

### Dark Mode

Android 跟随系统，Windows 支持用户手动切换（默认浅色）。

```
背景                 #121212
卡片背景             #1E1E1E
文字-主              #EFEFEF
文字-次              #AAAAAA
分割线               #2C2C2C
主色（暗色下）       #FF4D6A   略亮，保证对比度
```

> Dark Mode 下主色使用 `#FF4D6A`（Light 变体）而非 `#E8002D`，确保在深色背景上满足 WCAG AA 对比度要求（4.5:1）。

## 3. 字体规范

| 用途 | 字号 | 字重 | 颜色 |
|------|------|------|------|
| 搜索词（大词头） | 28sp | Bold | #1A1A1A |
| 音标 | 14sp | Regular | #E8002D |
| 词性标签 | 12sp | Medium | #E8002D |
| 中文释义 | 16sp | Regular | #1A1A1A |
| 英文释义 | 15sp | Regular | #333333 |
| 例句英文 | 14sp | Regular | #333333 |
| 例句中文 | 13sp | Regular | #666666 |
| 导航标签 | 12sp | Medium | — |
| 历史/收藏列表词 | 16sp | Regular | #1A1A1A |

字体栈：`Noto Sans SC`（中文）+ `Roboto`（英文/数字）

字号支持系统字体缩放（0.85x ~ 1.3x），使用 `textScaleFactor` 约束最大缩放比例，防止布局溢出。

## 4. 图标规范

- 风格：线性图标，2px 描边，圆角端点
- 尺寸：导航栏 24dp，操作按钮 20dp，列表辅助 18dp
- 来源：`material_symbols_outlined` 包 + 自定义 SVG

| 图标 | 用途 |
|------|------|
| search | 搜索 |
| volume_up | 发音（英式） |
| volume_down | 发音（美式） |
| star / star_border | 收藏/取消收藏 |
| history | 历史记录 |
| book | 词典 |
| translate | 翻译 |
| close | 清除搜索 |
| arrow_back | 返回 |
| more_vert | 更多操作 |
| settings | 设置 |
| download | 模型下载 |
| error_outline | 错误提示 |

## 6. 组件规范

### 搜索框

```dart
// 高度: 48dp, 圆角: 24dp
// 背景: #FFFFFF, 阴影: elevation 2
// 内边距: 水平16dp
// 光标颜色: #E8002D
```

### 词性标签 (PartOfSpeechChip)

```
背景: #FFF0F0, 圆角: 4dp
内边距: 水平8dp 垂直2dp
字号: 12sp, 颜色: #E8002D, 字重: Medium
```

### 发音按钮

```
圆形按钮, 直径36dp
图标: volume_up, 20dp
背景: #FFF0F0 (静止) → #E8002D (按下)
图标颜色: #E8002D (静止) → #FFFFFF (按下)
加载中: CircularProgressIndicator (strokeWidth: 2, color: #E8002D)
失败状态: 图标变为 error_outline, 颜色 #FF5252, tooltip "发音暂不可用"
```

### 收藏按钮

```
图标: star_border (未收藏, #999999)
      star (已收藏, #FFB800)
点击动画: scale 1.0 → 1.3 → 1.0, 200ms
```

### 例句卡片

```
背景: #FAFAFA, 左边框: 3dp solid #E8002D
内边距: 12dp
英文行: 14sp #333333
中文行: 13sp #666666, 上间距4dp
```

### 模型下载进度条

```
高度: 8dp, 圆角: 4dp
进度色: #E8002D
背景色: #E0E0E0
进度文字: 14sp #666666
```

## 7. 动效规范

| 场景 | 动效 | 时长 | 曲线 |
|------|------|------|------|
| 页面切换 | Slide from right | 300ms | easeInOut |
| 搜索框聚焦 | 展开 + 阴影加深 | 200ms | easeOut |
| 收藏点击 | Scale bounce | 200ms | elasticOut |
| 列表项出现 | Fade + slide up | 150ms | easeOut |
| Tab 切换 | Underline slide | 200ms | easeInOut |
| 发音按钮 | Ripple + color | 150ms | — |
| 模型下载进度 | 线性进度条动画 | 持续 | linear |

## 8. 空状态设计

| 场景 | 插图 | 文案 |
|------|------|------|
| 无搜索结果 | 放大镜+问号 | "未找到 "{word}"，试试其他词？" |
| 收藏为空 | 星星+书本 | "还没有收藏，查词时点击★收藏" |
| 历史为空 | 时钟 | "暂无搜索记录" |
| 词典加载中 | 进度环 | "正在初始化词典..." |
| AI 未安装 | 机器人图标 | "AI 功能未启用，前往设置下载模型" + [去设置] 按钮 |
| 模型下载失败 | 警告图标 | "下载失败，请检查网络后重试" + [重试] 按钮 |
