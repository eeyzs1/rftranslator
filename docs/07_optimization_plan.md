# 优化计划 — rftranslator

> **注意**：本文档记录当前和未来的优化方向。以源码为准。

## 1. 已完成的优化

### 1.1 翻译性能
- ✅ CTranslate2 FFI 直接调用，避免 HTTP 通信开销
- ✅ 翻译推理在独立 Isolate 中运行，不阻塞 UI
- ✅ 多模型并行翻译，结果独立展示
- ✅ 词典查询二分搜索，索引加载到内存

### 1.2 用户体验
- ✅ Material 3 / Fluent Design 双 UI 风格
- ✅ 12 种预设主题色 + 浅色/深色模式
- ✅ 中英双语国际化
- ✅ 历史记录长文本单行截断
- ✅ 词典查询自动回退到 AI 模型

### 1.3 构建与发布
- ✅ GitHub Actions CI/CD
- ✅ Inno Setup Windows 安装包（注册到系统搜索）
- ✅ ZIP 便携版
