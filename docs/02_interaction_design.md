# 交互设计方案 — rftranslator

> **注意**：本文档描述当前已实现的交互设计。以源码为准。

## 1. 全局交互约定

| 规则 | Android | Windows |
|------|---------|---------|
| 返回 | 系统返回键 / 左上角← | Alt+← / 左上角← |
| 右键菜单 | 长按弹出 BottomSheet | 右键弹出 ContextMenu |
| 键盘 | 软键盘自动弹出 | 物理键盘，Enter 触发翻译 |
| 导航 | 底部 NavigationBar（Material）/ 侧边 NavigationRail（Fluent） | 侧边 NavigationRail（Fluent）/ 底部 NavigationBar（Material） |

---

## 2. 翻译主页面交互流

```
启动
  │
  ▼
初始化页（加载模型管理器、词典管理器、预加载最近词典）
  │
  ▼
翻译主页面
  │ 输入文本 + 点击翻译按钮
  ▼
文本类型判断
  ├─ 单词/短语（≤3 词且 ≤50 字符）→ 词典查询
  │     ├─ 找到 → 显示词典结果（音标、释义、例句）
  │     └─ 未找到 → 回退到 OPUS-MT 模型翻译
  └─ 长句/段落 → OPUS-MT 模型翻译
```

---

## 3. 核心交互详细说明

### 3.1 翻译交互

**触发翻译的方式：**
1. 点击翻译按钮
2. 键盘 Enter 键

**语言选择：**
- 下拉选择源语言和目标语言
- 一键交换源语言和目标语言
- 支持 19 种语言

**翻译结果展示：**
- 词典结果：音标、词性、释义、例句（多词典结果并列展示）
- AI 翻译结果：OPUS-MT 模型翻译文本（多模型结果独立展示）
- 翻译进行中：显示加载指示器
- 词典未收录时自动回退到 AI 模型

### 3.2 收藏交互

```
点击收藏按钮
  → 切换收藏状态
  → 保存到 Hive 本地存储
```

### 3.3 历史记录交互

- 所有翻译自动保存到历史
- 按时间倒序显示
- 长文本仅显示首行（省略号截断）
- 支持删除历史记录

### 3.4 设置页交互

**UI 风格切换：**
```
选择 UI 风格：Material / Fluent / 自适应
  → 立即生效，无需重启
  → 持久化到 SharedPreferences
```

**主题模式：**
```
选择主题模式：浅色 / 深色 / 跟随系统
  → 立即生效
  → 两端均可手动切换
```

**翻译模型管理：**
```
未安装状态：
  → 显示 [下载] 按钮
  → 选择下载源（自动检测 / HuggingFace / ModelScope）

已安装状态：
  → 显示模型名称
  → 支持删除模型

支持从本地文件夹导入 CTranslate2 格式模型
```

## 4. 通知与反馈

使用 Toast 提示，场景包括：
- 复制成功
- 收藏/取消收藏
- 模型下载完成/失败
- 词典下载完成/失败

---

## 5. 加载状态设计

### 首次启动

初始化页加载模型管理器、词典管理器、预加载最近使用的 StarDict 词典。

### 翻译加载

- 词典查询：< 50ms，直接展示结果
- AI 模型翻译：显示加载指示器，翻译在独立 Isolate 中运行
- 多模型并行翻译时，每个模型结果独立展示，带进度指示

---

## 6. 错误处理交互

| 错误类型 | 展示方式 | 用户操作 |
|----------|----------|----------|
| 词典未找到 | 回退到 AI 模型翻译 | 自动处理 |
| TTS 播放失败 | 图标短暂变红 + 提示 | 可重试 |
| 模型下载失败 | 提示 + [重试] | 支持重试 |
| 存储空间不足 | Dialog 提示 | 引导清理存储 |
| CTranslate2 加载失败 | 设置页错误状态 | [重新加载] / [重新下载] |
