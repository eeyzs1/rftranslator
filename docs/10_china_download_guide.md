# 国内下载指南 — rftranslator

> **注意**：本文档描述当前的国内下载方案。以源码为准。

## 1. 问题背景

国内用户访问 Hugging Face 可能受到网络限制，导致模型和词典下载失败。rftranslator 提供了多种下载源选项来解决这个问题。

---

## 2. 下载源选项

### 2.1 模型下载源

在 **设置 → 翻译模型** 页面中，支持三种下载源：

| 下载源 | 说明 | 国内可用性 | 推荐场景 |
|--------|------|-----------|---------|
| **自动检测** | 先尝试 HuggingFace，失败自动回退 ModelScope | ✅ | 默认推荐 |
| **Hugging Face** | 官方源，模型最全 | ⚠️ 可能需要代理 | 有代理时 |
| **ModelScope** | 阿里云模型库，国内直连 | ✅ | 国内无代理时 |

### 2.2 词典下载源

词典下载使用 GitHub Release 作为源，国内可能需要代理。替代方案：
- 使用本地导入功能（MDict / StarDict）
- 通过代理下载

---

## 3. 使用建议

### 3.1 国内用户推荐设置

1. 打开 **设置 → 翻译模型**
2. 下载源选择 **ModelScope** 或 **自动检测**
3. 下载所需语言对的翻译模型

### 3.2 本地导入

如果网络下载不可用，可以使用本地导入：

**导入翻译模型：**
1. 在其他设备上下载 CTranslate2 格式模型
2. 将模型文件夹复制到本地
3. 在 **设置 → 翻译模型** 中点击 **从本地文件夹导入**

**导入词典：**
1. 在 **设置 → 词典管理** 中点击 **导入 MDict (.mdx)** 或 **导入 StarDict**
2. 选择本地词典文件

---

## 4. ModelScope 模型库

ModelScope（魔搭）是阿里云提供的模型托管平台，国内访问速度快。

模型下载 URL 格式：
```
https://modelscope.cn/models/{namespace}/{model_name}/resolve/master/{file_path}
```

rftranslator 的 ModelScope 下载逻辑会自动将 HuggingFace 模型路径转换为 ModelScope 路径。
